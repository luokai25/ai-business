# 🤖 MULTI-AGENT MONEY SYSTEM

## Running Agents

```bash
# Run all agents
python3 money-maker/run_all_agents.py

# Run in parallel (faster)
python3 money-maker/run_all_agents.py --parallel
```

## What Each Agent Does

### 1. 📦 Dropshipping Agent
- Finds trending products on Amazon
- Analyzes competition
- Calculates profit margins
- Identifies winning niches

### 2. 💰 Affiliate Agent
- Finds high-commission products
- Creates affiliate campaigns
- Tracks earnings
- Optimizes conversions

### 3. ✍️ Content Agent
- Creates social media posts (Twitter, Instagram, Facebook)
- Generates daily content calendar
- Writes ad copy

### 4. 🎯 Lead Agent
- Generates leads
- Qualifies prospects
- Creates outreach messages
- Reports on conversions
